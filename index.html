L'Objectif de l'application

Avoir un logiciel de gestion est très important pour surveillé au mieux votre commerce, mais celui-ci n'est pas facile d'accès a cause du coût élevé qu'il faut débourser pour l'avoir, c'est pour cela que nous vous aménons une solution efficace et accessible à absolument tous les bugdets, nous considérons qu'un élément aussi import doit être accessible à tous.

L'application est pensé pour être utilisé à tous les niveaux de l'échelle du commerce, que ça soit du stock dépôt à la mise en vente dans le magasin, notre application vous accompagnera toujours peu importe la taille de votre commerce, notre leitmotive est d'être tout aussi bien à côté du boulangé que du super marché, tout comme les alimentations, les bijoutiers, les vendeurs de vêtements, les quincailléries, patissiers, libraires, dépôt de boisson en tout genre ...

Pourquoi utiliser l'Application KoTeka ?

Les différentes fonctionnalités de l’application

L'application presente un bon nombre de fonctionnalités, pour des besoins d'exhaustivité, nous vous conseillons de consulter la section tutoriels ou vous aurez accès à une présentation global. Mais laissez nous vous présenter une infime partie des fonctionnalité que comporte notre application.

Des fiches clients et fournisseurs pour une meilleur connaitre d'où vient et où va le produit.

Vous aurez toujours un oeil sur qui est-ce qui vous doit, combien il vous doit et qu'est-ce qu'il a pris pour cette dette.

Vous pourrez organiser vos produit selon vos thématique : couleur, taille, prix, marque, modèle, la provenance...

Gestion de stock

Vous aurez une vision plus détaillé de chacun de vos stock.

LICENCE

Après 1 mois d'utilisation gratuit qui vous est offert lors de votre inscription, pour continuer à utiliser notre application il va falloir payer une licence renouvellable. Elle est presenté sous trois formules en fonction des pays.
Cependant, avant de vous présenter les formules disponibles, laissez nous vous parler de notre programme de filliation (parrainage) avec lequel vous pouvez carrement utiliser notre application sans débourser un seul centime.

3 recommandations = 1 mois d'utilisation gratuit (profitez-en).

Une fois inscript, un code parent (code de parrainage) vous sera attribué.
Si notre application vous est utile et que vous la recommandez à quelqu'un d'autre, donnez à ce dernier votre code parent afin qu'il l'insert dans le champs code parent lors de l'inscription, ainsi vous bénéficirait de 10 jours d'utilisations gratuits et lui aussi bénéficira de 10 jours d'utilisation gratuits.
Vous aurez compri qu'un recommandation vous profite ainsi qu'a celui que vous avez recommandé, donc, partagez sans modération !
